# AI Powered Smart Health Monitoring & Early Risk Detection System

## Team Project

This project develops an intelligent healthcare monitoring system that collects physiological data and applies machine learning models to detect abnormal health patterns.

## Objectives

- Monitor real-time health parameters
- Apply AI for anomaly detection
- Provide medical alerts
- Store patient health history
- Display data on dashboard

## System Components

1. IoT Health Data Collection
2. Cloud Database
3. Machine Learning Prediction Model
4. Web Monitoring Dashboard
